# 🔥 Cue Barbecue — Digital Smokehouse Experience

> **Scratch-Made Barbecue. Built for Gathering.**

A world-class, premium restaurant website for Cue Barbecue — Metro Atlanta's scratch-made barbecue destination with three locations (Milton, Cumming, Peachtree Corners).

## ✨ Features

### Premium Design System
- **Deep charcoal/smoked black** base with warm bone/parchment tones
- **Burnt-orange ember** accents with brass/copper highlights
- Sharp typography using Playfair Display, Oswald, Lato, and Cormorant Garamond
- Controlled premium motion system with no gimmicks

### Interactive Experiences
- **🔥 Ember Particle System** — Canvas-based floating ember particles in the hero section
- **🌙 Smokehouse Mode** — Day/Night toggle that transforms the site's mood
- **🎯 Flavor Map** — Interactive flavor wheel that recommends dishes based on taste preferences
- **❓ "What Should I Order?" Quiz** — Two-question personality quiz for first-time visitors
- **📜 Smoke Scroll Line** — Animated scroll progress indicator along the left edge
- **🔲 Fire to Table** — Horizontal draggable storytelling section showing the 5-stage BBQ process

### Conversion-Focused Sections
- **Cinematic Hero** with ember particles, strong headline, and 4 primary CTAs
- **Location Chooser** — "Choose Your Smokehouse" with live badges, hours, and direct ordering
- **Intent Paths** — "I'm Here To..." split-path routing by visitor intent
- **Signature Dish Cards** — Horizontally scrollable spotlight cards with hover animations
- **Premium Menu Explorer** — Tabbed, categorized menu with badges and descriptions
- **Catering Microsite** — Three-tier packages + inline quote request form
- **Live Music Section** — Featured artist, upcoming calendar, animated equalizer bars
- **Hickory Room** — Private event booking section with event types and gallery
- **Gift Cards** — Interactive card mockup with amount selector
- **Cue Standard** — Craftsmanship wall highlighting scratch-made commitment
- **Our Story** — Brand narrative with animated counter stats
- **Newsletter** — "Join the Pit Crew" email capture

### Technical Features
- **Zero dependencies** — Pure HTML, CSS, and vanilla JavaScript
- **Mobile-first responsive** design with thumb-zone CTAs
- **Scroll-triggered reveal animations** using IntersectionObserver
- **Sticky navigation** with scroll-aware background blur
- **Sticky order orb** with subtle pulse animation
- **Full-screen coal loader** animation on site entry
- **Parallax effects** and counter animations
- **ARIA labels** and semantic HTML for accessibility

## 🚀 Getting Started

Simply open `index.html` in a browser. No build step required.

```bash

open index.html

# Option 2: Local server
npx serve .
```

## 🎨 Design Philosophy

The site is built around five principles:
1. **Appetite** — Make the food impossible to ignore
2. **Atmosphere** — Make the space feel alive
3. **Utility** — Make ordering and booking effortless
4. **Brand** — Make Cue feel distinct, not generic barbecue
5. **Memory** — Give visitors moments they've never seen on a restaurant site
